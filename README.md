# Layoffs Data Analysis

## Overview

This project explores worldwide layoffs using SQL and Power BI to identify trends across industries, countries, funding stages, and companies.

---

## Business Problem

Business leaders wanted to understand:

- Which companies had the largest layoffs?
- Which industries were most affected?
- Which countries experienced the highest layoffs?
- How did layoffs change over time?

---

##  Dataset

Global Layoffs Dataset

Includes:

- Company
- Industry
- Country
- Total Laid Off
- Percentage Laid Off
- Funding Stage
- Date

## Key Insights

- Technology industry experienced the highest layoffs.
- The United States recorded the largest number of layoffs.
- Layoffs peaked during major economic downturns.
